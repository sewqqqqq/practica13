# 📌 Проект: Task Management Information System Database

## Описание предметной области

Данная база данных предназначена для информационной системы управления задачами.  
Система позволяет:

- создавать и распределять задачи
- отслеживать статусы выполнения
- управлять приоритетами
- классифицировать задачи по категориям
- разграничивать доступ пользователей по ролям

База данных является централизованным хранилищем информации для веб или корпоративной системы управления проектами.

---

## Основные сущности

### Таблицы-справочники:
- roles (роли пользователей)
- users (пользователи системы)
- categories (категории задач)
- statuses (статусы задач)
- priorities (приоритеты задач)

### Таблица фактов:
- tasks (основная сущность системы)

---

## Роли доступа

- Пользователь — выполнение задач
- Менеджер — распределение задач
- Администратор — управление системой

---

## Тестовые данные

- Users: 30 записей  
- Categories: 20 записей  
- Statuses: 20 записей  
- Priorities: 20 записей  
- Tasks: 100 записей
