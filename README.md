# Личная медиатека (Media Library System)

## Название проекта

13practicaa — база данных личной медиатеки

---

## Описание предметной области

База данных предназначена для организации личной медиатеки пользователя.

Система позволяет хранить и анализировать информацию о:

- фильмах
- сериалах
- книгах
- музыке
- видеоиграх

Пользователь может:

- вести список просмотра
- хранить оценки и отзывы
- отслеживать прогресс просмотра
- анализировать предпочтения
- хранить изображения медиаобъектов

---

## Основные сущности базы данных

### media_items

Главная таблица системы.

Хранит:

- название медиаобъекта
- тип контента
- год выпуска
- жанр
- платформу
- статус просмотра
- изображение (BLOB)

Типы контента:

- фильм
- сериал
- книга
- музыка
- игра

---

### genres

Справочник жанров:

- боевик
- драма
- фантастика
- комедия
- ужасы
- аниме и др.

---

### platforms

Платформы и носители:

- Netflix
- Steam
- Spotify
- DVD
- Blu-Ray
- Xbox
- PlayStation и др.

---

### statuses

Статусы просмотра:

- Хочу посмотреть
- В процессе
- Просмотрено

---

### users

Информация о пользователях системы.

---

### reviews

Отзывы и оценки пользователей.

Позволяют:

- ставить рейтинг
- хранить комментарии

---

## Бизнес-процессы

### Импорт метаданных

Система предусматривает возможность интеграции с внешними сервисами:

- TMDB
- OpenLibrary

---

### Отслеживание прогресса

Пользователь может изменять статус просмотра контента.

---

### Формирование рекомендаций

Рекомендации строятся на основе:

- жанров
- пользовательских оценок
- истории просмотров

---

### Формирование списков

Пользователь может создавать списки:

- хочу посмотреть
- просмотрено
- избранное

---

### Статистика медиапотребления

Система позволяет анализировать:

- популярные жанры
- используемые платформы
- количество просмотренного контента
