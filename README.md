# Личная медиатека (Media Library System)

## Название проекта
13practicaa — База данных личной медиатеки

---

## Описание предметной области

База данных предназначена для организации личной медиатеки пользователя, включающей:

- фильмы
- сериалы
- книги
- музыку
- видеоигры

Система позволяет хранить, структурировать и анализировать медиаконтент, а также отслеживать предпочтения пользователя.

---

## Основные сущности базы данных

### 📌 Медиаобъекты (media_items)
Основная таблица системы, содержащая:
- название
- тип (фильм, сериал, книга, игра, музыка)
- год выпуска
- жанр
- платформа
- статус просмотра

---

### Жанры (genres)
Классификация контента:
- боевик
- драма
- комедия
- фантастика и др.

---

### Платформы (platforms)
Места хранения или просмотра:
- Netflix
- Steam
- книги
- DVD
- Spotify и др.

---

### Статусы просмотра (statuses)
- Хочу посмотреть
- В процессе
- Просмотрено

---

### Пользователи (users)
Хранят информацию о пользователях системы.

---

### Отзывы и оценки (reviews)
Позволяют:
- ставить рейтинг (1–10)
- оставлять комментарии

---

## Бизнес-процессы

### Импорт метаданных
Возможность интеграции с внешними источниками (TMDB, OpenLibrary).

---

### Отслеживание прогресса
Пользователь может отслеживать статус:
- хочу посмотреть
- в процессе
- просмотрено

---

### Рекомендации
Формируются на основе:
- жанров
- оценок пользователей
- истории просмотров

---

### Формирование списков
Пользователь формирует список контента для просмотра.

---

### Статистика
Анализ медиапотребления:
- по жанрам
- по платформам
- по типу контента

---

## Тестовые данные

- media_items — 100 записей  
- genres — 20 записей  
- platforms — 20 записей  
- reviews — 100 записей  
- users — 3 записи  
- statuses — 3 записи  

---

## Технологии

- Microsoft SQL Server (SSMS)
- T-SQL
- циклы WHILE
- FOREIGN KEY связи
