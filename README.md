## Описание проекта

Проект представляет собой WPF-приложение для хранения и управления личной медиатекой пользователя.

Система позволяет работать с различными типами медиаконтента:

- фильмы
- сериалы
- книги
- музыка
- видеоигры

Приложение взаимодействует с базой данных Microsoft SQL Server и предоставляет удобный визуальный интерфейс для просмотра и управления контентом.

---

# Описание выполненной работы

В рамках практической работы была доработана клиентская часть информационной системы.

Реализован полноценный CRUD-функционал:

- Create — добавление записей
- Read — просмотр записей
- Update — редактирование записей
- Delete — удаление записей

---

# Реализованные возможности

## 1. Каталог медиаконтента

Создана страница `CatalogPage.xaml`.

Функционал страницы:

- загрузка данных из базы данных
- отображение медиаконтента в виде карточек
- использование `ListView`
- использование `WrapPanel` для отображения сетки карточек
- адаптивное расположение элементов

---

## 2. Карточки медиаконтента

Каждая карточка содержит:

- изображение медиаконтента
- название
- жанр
- статус просмотра
- рейтинг
- сложность
- краткое описание

При отсутствии изображения используется локальная заглушка `film.png`.

---

## 3. CRUD-операции

Реализованы:

### Добавление записей
Пользователь может добавить новый медиаобъект через форму `AddEditPage.xaml`.

### Редактирование записей
Доступно изменение информации о выбранном объекте.

### Удаление записей
Перед удалением отображается подтверждение действия.

---

## 4. Поиск

Реализован поиск по:

- названию
- описанию

Список обновляется автоматически при вводе текста.

---

## 5. Фильтрация

Добавлена фильтрация по:

- статусу
- категории/жанру

Используется `ComboBox`.

---

## 6. Сортировка

Реализована сортировка:

- по названию
- по рейтингу
- по статусу

Поддерживается:

- сортировка по возрастанию
- сортировка по убыванию

Список обновляется без перезапуска приложения.

---

## 7. Работа с изображениями

Реализована поддержка изображений:

- загрузка изображения с компьютера
- хранение изображений в БД (`VARBINARY(MAX)`)
- отображение изображений в WPF
- отображение превью
- использование заглушки при отсутствии изображения

---

## 8. Авторизация и навигация

Реализована навигация между окнами:

- LoginPage
- RegisterPage
- CatalogPage
- AddEditPage

После успешной авторизации пользователь переходит в каталог.

# Структура базы данных

## Основные таблицы

### media_items
Основная таблица медиаконтента.

Содержит:

- название
- жанр
- статус
- изображение
- описание
- рейтинг

---

### genres
Таблица жанров медиаконтента.

---

### statuses
Таблица статусов просмотра.

Примеры:

- Просмотрено
- В процессе
- Отложено
- Хочу посмотреть

---

### users
Таблица пользователей системы.

---

### reviews
Отзывы и оценки пользователей.
