# MediaLibraryApp

## Описание выполненной работы
Разработана клиентская часть информационной системы для личной медиатеки на основе базы данных из практической работы №13. В WPF-приложении реализованы два основных окна: `LoginPage.xaml` для входа в систему и `RegistrationPage.xaml` для регистрации нового пользователя. Интерфейс оформлен в едином фирменном стиле с централизованным описанием цветов, шрифтов и стилей в `App.xaml`. В проект также добавлены логотип в формате `.png` и иконка приложения в формате `.ico`.
