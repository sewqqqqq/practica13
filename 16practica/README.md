# MediaLibraryApp (14practica)

## Название проекта
Информационная система "Личная медиатека": клиентское WPF-приложение для авторизации, регистрации и просмотра каталога медиаконтента.

## Описание выполненной работы
В рамках практической работы №14 реализована клиентская часть приложения:
- Добавлена страница `CatalogPage.xaml` с визуализацией сущностей в виде карточек.
- Для вывода карточек используется `ListView` с нестандартной панелью элементов `WrapPanel` (сетка карточек).
- Карточка содержит изображение (из BLOB-поля `media_items.image`, при ошибке/отсутствии изображения отображается заглушка), название, краткое описание, статус, рейтинг и уровень сложности.
- Реализована загрузка данных из базы `13practicaa` (`media_items` + `genres` + `statuses`) через ADO.NET.
- Добавлена навигация после успешного входа и после регистрации на страницу каталога.
- Сохранено оформление по ресурсам и цветовой палитре из `App.xaml`.
